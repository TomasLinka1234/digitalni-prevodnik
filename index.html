MU/TH/UR 6000 INTERFACE

Digitální převodník

Jedna stránka pro převody číselných soustav, pochopení IP adres a zápis barev pomocí hexadecimálního kódu.

SoustavyBinární, decimální a hexadecimální převod s limitem do 32 bitů.
IP adresyIPv4, binární zápis, maska, prefix, síťová a broadcast adresa.
BarvyZápis RGB a HEX, binární kanály a náhled výsledné barvy.

Převod mezi soustavami

Zadej číslo do binární, decimální nebo hexadecimální soustavy. Limit je 32 bitů, tedy maximálně 4294967295.

Jak postupovat
  • Zadej číslo do jedné soustavy.
  • Systém ověří povolené znaky a délku vstupu.
  • Vstup se převede na decimální hodnotu.
  • Z ní se dopočítá binární i hex zápis.
Stručné vysvětlení
  • Binární soustava používá jen 0 a 1.
  • Decimální používá číslice 0 až 9.
  • Hexadecimální používá 0–9 a A–F.
  • Hex je praktický zkrácený zápis binárních dat.
Limity
  • Binární vstup: max. 32 znaků.
  • Hex vstup: max. 8 znaků.
  • Decimální vstup: 0 až 4294967295.

      Barvy pomocí soustav

      Na webu se barvy zapisují jako HEX ve tvaru #RRGGBB. Každý kanál má hodnotu 0 až 255, tedy 00 až FF v hexu.

      Hexadecimální výstup—

      Jak postupovat
      • Zadej HEX nebo RGB.
      • Každý kanál se převede mezi 0–255 a 00–FF.
      • Ze tří kanálů vznikne výsledná barva.
      Příklady
      • #FF0000 = červená
      • #00FF00 = zelená
      • #0000FF = modrá
      Stručné vysvětlení
      • RR = red
      • GG = green
      • BB = blue
      • Každý pár je hex zápis hodnoty 0–255.
